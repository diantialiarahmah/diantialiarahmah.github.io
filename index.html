Tentang Smart Incubator Monitoring System

Smart Incubator Monitoring System adalah sistem pemantauan cerdas untuk mengawasi kualitas suhu dan kelembapan inkubator bayi secara real-time menggunakan Arduino Uno, sensor DHT11, dan buzzer.

Sensor DHT11 membaca kondisi lingkungan di dalam inkubator dan data diproses oleh Arduino Uno. Apabila suhu atau kelembapan berada di luar rentang aman, sistem akan mengaktifkan buzzer sebagai alarm peringatan dini.

Sistem ini dirancang sebagai solusi sederhana, ekonomis, dan andal yang dapat dikembangkan lebih lanjut ke sistem monitoring terpusat berbasis ROS 2 dalam ekosistem Smart Hospital.

Pentingnya Smart Incubator Monitoring System

Pengaturan Suhu dan Kelembapan

Bayi, terutama bayi prematur, belum mampu mengatur suhu tubuhnya sendiri. Inkubator berfungsi menjaga suhu dan kelembapan tetap stabil agar bayi aman dan nyaman.

Risiko Kondisi Tidak Stabil

Kondisi inkubator yang tidak stabil dapat menyebabkan gangguan pernapasan, iritasi kulit, kehilangan cairan tubuh, hingga meningkatkan risiko infeksi.

Keterbatasan Pemantauan Manual

Pemantauan manual bersifat berkala dan sangat bergantung pada ketelitian manusia sehingga berisiko terjadi keterlambatan penanganan.

Sistem Otomatis Berbasis Mikrokontroler

Sistem otomatis memungkinkan pemantauan terus-menerus dan memberikan peringatan dini secara cepat melalui alarm.

Fitur Sistem

Tahap I - Environment Setup

Tahap ini bertujuan untuk menyiapkan lingkungan kerja perangkat lunak agar sistem dapat dikembangkan, dijalankan, dan diintegrasikan dengan ROS 2 secara optimal.

    1️⃣ Menyiapkan Sistem Operasi Utama (Windows 11)

    Windows 11 digunakan sebagai host system (sistem utama) untuk menjalankan seluruh pengembangan.

    Fungsi Utama:

    - Menjalankan Arduino IDE

    - Menjalankan Docker Desktop

    - Menjalankan WSL 2 sebagai jembatan ke Linux

    Pastikan

    - Windows 11 sudah ter-update

    - Virtualization aktif di BIOS

    - RAM minimal 8 GB (disarankan)

    2️⃣ Mengaktifkan WSL 2 (Windows Subsystem for Linux)

    WSL 2 memungkinkan kita menjalankan Linux langsung di dalam Windows tanpa dual boot.

    Langkah-langkah:

    - Buka PowerShell (Run as Administrator)

    - Jalankan perintah:

    wsl --install

    - Restart komputer

    - Pastikan WSL versi 2:

    wsl --set-default-version 2

    3️⃣ Instalasi Ubuntu 24.04 LTS (OS Guest)

    4️⃣ Update Sistem Ubuntu

    5️⃣ Instalasi ROS 2 Jazzy Jalisco

    6️⃣ Instalasi Docker Desktop

    7️⃣ Menjalankan micro-ros-agent (Docker)

    8️⃣ Instalasi Arduino IDE

    9️⃣ Instalasi Python (untuk ROS 2 Node)

Contoh Program Arduino

#include 

#define DHTPIN 2        // Pin DATA DHT11
#define DHTTYPE DHT11
#define BUZZER 8        // Pin buzzer

DHT dht(DHTPIN, DHTTYPE);

void setup() {
  Serial.begin(9600);
  dht.begin();
  pinMode(BUZZER, OUTPUT);

  Serial.println("=== Monitoring Inkubator Bayi ===");
}

void loop() {
  float kelembapan = dht.readHumidity();
  float suhu = dht.readTemperature(); // Celcius

  // Cek jika sensor gagal membaca
  if (isnan(suhu) || isnan(kelembapan)) {
    Serial.println("Gagal membaca sensor DHT11!");
    digitalWrite(BUZZER, HIGH);
    delay(1000);
    digitalWrite(BUZZER, LOW);
    delay(1000);
    return;
  }

  // Tampilkan data
  Serial.print("Suhu: ");
  Serial.print(suhu);
  Serial.print(" °C | Kelembapan: ");
  Serial.print(kelembapan);
  Serial.println(" %");

  // Logika alarm inkubator bayi
  if (suhu < 32 || suhu > 36 || kelembapan < 50 || kelembapan > 70) {
    Serial.println("⚠️ PERINGATAN: Kondisi TIDAK AMAN!");
    digitalWrite(BUZZER, HIGH);
  } else {
    Serial.println("✅ Kondisi AMAN");
    digitalWrite(BUZZER, LOW);
  }

  Serial.println("----------------------------");
  delay(2000);
}

Troubleshooting

  • Arduino gagal dikompilasi → Periksa library dan board
  • Node ROS 2 tidak terhubung → Cek port & micro-ROS agent
  • Perintah ROS 2 tidak dikenali → Source environment ROS 2
